HTML (HyperText Markup Language) არის სტანდარტი, რომელსაც იყენებენ ვებ-ბრაუზერებისთვის გვერდების სტრუქტურის განსაზღვრაში: სათაურები, პუნქტები, იუმაღალი ტექსტი, სურათები, ბმულები და ფორმები.

> ისტორია > მარკაპი WHATWG Living Standard

History

HTML შეიქმნა 1990–1993 წლებში ვების დამფუძნებლის ტიმ ბერნერს-ლი მიერ. მას შემდეგ ის განვითარდა HTML 2, HTML 3.2, HTML 4 და შემდეგ HTML5/WHATWG Living Standard-ს მიმართულებით.

აქ მოკლე ქუთაისური ფაქტები და მენიუ — ეს გვერდი წააგავს Wikipedia-ის სტრუქტურას, მაგრამ ნავიგაციისა და მარტივი დიზაინისთვის არის გამარტივებული.

Markup

HTML იყენებს ელემენტებს — <h1>, <p>, <a>, <img>, <ul> და სხვა — რომლებიც აღწერენ ვებ-დოკუმენტის სხვადასხვა ნაწილებს. ელემენტები შეიძლება ჰქონდეთ ატრიბუტები, როგორიცაა src ან href, რომლებიც განსაზღვრავენ წყაროს ან ბმულის მისამართს.

<h1>სათაური</h1>
<p>პარაგრაფი</p>
<img src="image.jpg" alt="სურათი"/>

Semantic HTML

სემანტიკური HTML აძლევს მნიშვნელობას ელემენტებს (მაგ. <article>, <nav>, <header>), რაც ეხმარება მოძიების სისტემებსა და პრეაქტორებს (screen readers) სწორი ინტერპრეტაციის გაკეთებაში.

ქვემოთ ორი ღილაკი: პირველი მიბმულია გარეთ (Wikipedia), მეორე — გამოხატული JavaScript-ით რათა აჩვენოს გამოფენა.
