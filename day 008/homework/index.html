Web Development ნიშნავს ვებსაიტების ან ვებაპლიკაციების შექმნას და განვითარებას. ეს მოიცავს ყველაფერს — დიზაინიდან დაწყებული, ფუნქციონალობამდე (როგორ მუშაობს საიტი „უკანა მხარეს“)

Front-End — ეს არის ის ნაწილი, რასაც მომხმარებელი ხედავს (დიზაინი, ტექსტები, ღილაკები, სურათები). Front-End-ისთვის ვიყენებთ ძირითადად HTML, CSS, JavaScript.

Back-End — ეს არის ის ნაწილი, რაც თვალით არ ჩანს, მაგრამ ამუშავებს მონაცემებს. მაგალითად, როდესაც მომხმარებელი შედის საიტზე, რეგისტრირდება ან აკეთებს შეკვეთას. Back-End-ისთვის იყენებენ Python, PHP, Node.js, SQL და სხვა ენებს.

HTML (HyperText Markup Language) — არის ვებსაიტების საწყისი ენა მას ვიყენებთ იმისთვის, რომ საიტის სტრუქტურა შევქმნათ — ანუ რას შეიცავს საიტი: ტექსტს, სურათებს, ბმულებს და ა.შ.

Tag (ტეგი) — არის HTML-ის ძირითადი ელემენტი, რომელიც განსაზღვრავს რა ტიპის შინაარსია გვერდზე.

როდესაც ფოტოს ვიწერთ და ვამატებთ ფაილად, ის ყოველთვის იმუშავებს, თუნდაც ინტერნეტში ის ბმული წაიშალოს.თუ ფოტოს მხოლოდ მისამართით ვამატებთ, ის გაქრება, როცა იმ ბმულზე არსებული სურათი წაიშლება ან შეიცვლება.
