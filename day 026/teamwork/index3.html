HTML კოდირების ენა არის სტანდარტული მარკირების ენა, რომელიც გამოიყენება ინტერნეტში შინაარსის შესაქმნელად და სტრუქტურირებისთვის. იგი ემსახურება როგორც ფუნდამენტური სამშენებლო ბლოკი ყველა ვებგვერდისთვის, იყენებს "ტეგებს" და "ელემენტებს", რათა ვებ-ბრაუზერებს უთხრან, როგორ აჩვენონ ტექსტი, სურათები და სხვა მულტიმედია.

HTML არის HyperText Markup Language-ის შემოკლება.

    ეს არის მარკირების ენა, რომელიც გამოიყენება ვებგვერდების შესაქმნელად.
    HTML-ის საშუალებით განისაზღვრება ვებგვერდის სტრუქტურა, ანუ რა ელემენტები იქნება გვერდზე: სათაურები, ტექსტი, სურათები, ბმულები, სიები და სხვა. ბრაუზერი კითხულობს HTML-ს და აჩვენებს მომხმარებელს ვებგვერდს შესაბამისი სახით.
    HTML არ არის პროგრამირების ენა, რადგან მას არ შეუძლია ლოგიკური ამოცანების შესრულება. მისი მთავარი დანიშნულებაა ინფორმაციის სწორად დალაგება და წარმოდგენა.
    HTML ხშირად გამოიყენება სხვა ტექნოლოგიებთან ერთად. CSS გამოიყენება ვებგვერდის დიზაინისთვის, ხოლო JavaScript — გვერდის ინტერაქტიულობისთვის.
