about css

CSS მნიშვნელოვანია, რადგან ის არის ვებ დიზაინის ენა, რომელიც პასუხისმგებელია ვებსაიტის გარეგნობაზე, განლაგებაზე და ვიზუალურ მიმზიდველობაზე. ის განასხვავებს დიზაინს შინაარსისგან (HTML), რაც ვებსაიტებს უფრო ეფექტურს, ხელმისაწვდომს და ადაპტირებულს ხდის სხვადასხვა მოწყობილობაზე.

CSS-ს შეუძლია:

    ტექსტის ფერის, ზომის და შრიფტის შეცვლა
    ფონის ფერის ან სურათის დამატება
    ელემენტების ზომის, მდებარეობის და განლაგების კონტროლი
    გვერდის სტრუქტურის ლამაზად დალაგება (მაგალითად, სვეტებად)
    ღილაკების, მენიუს და სხვა ელემენტების გაფორმება
    ანიმაციების და გადასვლების (transition) დამატება
    ვებგვერდის მორგება სხვადასხვა ეკრანის ზომაზე (responsive design)
HTML ქმნის გვერდის შინაარსს, ხოლო CSS აძლევს მას ვიზუალურ იერს.
