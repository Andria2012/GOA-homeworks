სიის და ელემენტების მაგალითი

ამ დოკუმენტში წარდგენილია: ჰედინგები, პარაგრაფები, ღილაკები, დალაგებული და დაულაგებელი სიები, ასევე აღწერილობითი სია.

1. ჰედინგები და პარაგრაფები

მ.Level 3 ჰედინგი

ეს არის მოკლე აბზაცი — შეგიძლიათ შეცვალოთ ტექსტი თქვენი მიზნების მიხედვით.

მეორე აბზაცი: აქ ვაჩვენებთ კიდევ ერთ მაგალითს, სადაც არის ვრცელი ტექსტი და ფორმატირება.

2. დალაგებული სიები (3 ტიპი)

1) რიცხვების ჩამონათვალი

  1. პირველი ელემენტი
  2. მეორე ელემენტი
  3. მესამე ელემენტი

2) ABC (ანბანის მიხედვით)

  1. ელემენტი A
  2. ელემენტი B
  3. ელემენტი C

3) რომაული ციფრებით (I, II, III...)

  1. პირველი (I)
  2. მეორე (II)
  3. მესამე (III)

3. დაულაგებელი სიები (nested lists)

κάθε დაულაგებელ სიას აქვს შიდა დაულაგებელი სია (nested <ul>).

  • მთავარი პუნქტი A
    • ქვეპუნქტი A.1
    • ქვეპუნქტი A.2
  • მთავარი პუნქტი B
    • ქვეპუნქტი B.1
    • ქვეპუნქტი B.2
      • გა-ქვეპუნქტი B.2.a
  • მთავარი პუნქტი C
    • ქვეპუნქტი C.1
    • ქვეპუნქტი C.2

4. აღწერილობითი სია (Description List)

აღწერილობითი ელემენტები გამოიყენება ტერმინებისა და მათი განმარტებების წყვილებისათვის.

HTML
ჰიპერტექსტი მარკუპის ენა — ვებ-გვერდების შექმნის ძირითადი ენა.
CSS
სტილებისა და ხედვის განსაზღვრა — როგორ გამოიყურება HTML ელემენტები.
JavaScript
დინამიური ქცევა და ინტერაქტიურობა ვებ-გვერდზე.

დოკუმენტი მართვიებულია მომხმარებლის მოთხოვნის მიხედვით — შეიცავს ყველა მოთხოვნილ ელემენტს.
